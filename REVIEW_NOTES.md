## 原聊天室

原项目实现了 C++ TCP 客户端、服务端和线程池，但依赖仓库中不存在的 `myhead.h`，并且没有处理 TCP 短读、粘包、部分写入和异常断线时的在线列表并发修改。终端客户端也不能满足前端窗口需求。

整合版保留登录、退出、在线列表、广播和并发服务语义，改为浏览器可访问的 HTTP/JSON 协议。共享状态由 `std::mutex` 保护，每个 HTTP 连接由独立线程处理。

## 正确词典项目

`ggg0057/dictionary` 包含完整 C++/TCP/SQLite 设计及 7,987 行词库。原服务端同样依赖 Linux `myhead.h`、POSIX Socket 和外部 SQLite 开发库，无法直接在当前 Windows 环境构建；原 TCP 结构体协议还存在短读和字符串终止风险。

整合版复用其原始 `dict.txt` 和服务端查询语义。为提供无需安装依赖的 Windows 程序，词库启动时直接加载到只读哈希表，去重后为 7,975 个词条；机器人支持英文精确查询与中文释义反向查询。
